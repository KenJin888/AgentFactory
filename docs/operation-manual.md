# AgentFactory - 用户操作手册

## 1. 简介
智能体工厂是一套用于创建、管理、发布和使用 AI 智能体的可视化平台，同时提供知识库、技能、工具和系统管理能力。

### 主要功能亮点
- 统一管理智能体全生命周期：创建、编辑、发布、下线、复制。
- 内置知识库能力：支持数据集、文档、切片、检索测试。
- 支持技能与 MCP 工具管理：可创建、导入、导出、维护。
- 提供系统级后台：用户、角色、菜单、部门、岗位、字典、参数等管理。
- 提供数据统计与监控：Token 报表、系统概况、服务器监控。

### 适用人群或使用场景
- 你需要快速搭建团队可复用的智能体能力。
- 你需要运营内部智能体广场并管理发布版本。
- 你需要通过权限体系进行多角色协同和系统治理。

## 2. 核心功能使用说明

### 2.1 账号登录与注册
**功能描述**  
用于账号登录、注册新用户、忘记密码后重置密码。

**操作步骤**
1. 打开登录页，输入用户名和密码，点击`登 录`。
2. 如果你是新用户，点击`马上注册`，填写用户名、密码、确认密码后提交。
3. 如果你忘记密码，点击`忘记密码?`，输入用户名和新密码进行重置。
4. 登录成功后，系统会自动跳转到首页或你之前访问的页面。

**注意事项和常见问题**
- 注册与重置密码时，两次密码必须一致。
- 登录失败时，优先检查账号密码是否正确，或后端服务是否可用。
- 重置密码成功后，请使用新密码重新登录。

### 2.2 智能体广场
**功能描述**  
用于发现和使用公开智能体，并支持收藏、复制、维护（创建者）。

**操作步骤**
1. 进入`智能体广场`后，可通过搜索框输入关键词筛选智能体。
2. 点击分类标签切换类型，或开启`收藏`筛选查看已收藏项。
3. 点击智能体卡片进入会话，系统会自动创建会话并跳转到聊天页。
4. 点击复制操作：
   - 你是创建者时，可选`复制全新副本`或`复制可更新版本`。
   - 非创建者时，复制后生成独立副本到`我的空间`。
5. 创建者可打开维护弹窗，更新广场信息或执行`下线`。

**注意事项和常见问题**
- 会话页签数量有上限，超过后会提示无法继续打开。
- 下线操作会影响其他用户可见性，请谨慎执行。


### 2.3 我的空间
**功能描述**  
用于管理你的私有智能体、查看会话历史与 Token 消耗。

**操作步骤**
1. 点击`创建智能体`，可选择：
   - `新智能体`：打开创建弹窗；
   - `从文件导入`：上传 JSON 配置；
   - `智能创建`：输入需求（当前界面已预留，后续能力扩展）。
2. 在`我的智能体`列表中，可执行`编辑`、`发布`、`删除`。
3. 点击某个智能体卡片，可直接创建新会话并进入聊天。
4. 在`会话历史`中，可按分页查看历史会话消耗记录，点击某行可跳转对应会话。

**注意事项和常见问题**
- 导入 JSON 必须包含关键字段（如 `name`、`prompt_template`）。
- 发布前建议先在对话页验证提示词与模型配置是否符合预期。


### 2.4 对话会话
**功能描述**  
用于与智能体持续对话，支持模型切换、历史会话、文件上传、语音输入。

**操作步骤**
1. 在输入框输入问题，点击发送。
2. 如需上传文件，先选择文件后再发送消息。
3. 点击顶部`历史记录`可切换/删除历史会话。
4. 点击`新对话`可基于当前配置创建新的会话。
5. 对话生成中可点击`停止`中断流式输出。

**注意事项和常见问题**
- 若会话不存在或加载失败，请检查会话链接是否有效。


### 2.5 知识库
**功能描述**  
用于管理数据集与文档，并进行切片检索测试，支撑智能体 RAG 能力。

**操作步骤**
1. 点击左侧`+`新建数据集，填写名称和描述。
2. 选中数据集后，点击`上传文件`上传文档。
3. 在文档列表中可执行`重新解析`、`停止解析`、`下载`、`重命名`、`删除`。
4. 点击文档`切片`进入 Chunk 管理，可新增、编辑、删除切片。
5. 点击`检索测试`，输入问题并设置 `top_k`，查看命中切片结果。
6. 如果你是数据集管理者，可在数据集菜单中设置权限。

**注意事项和常见问题**
- 只读权限下可查看与检索，但不能修改知识库内容。
- 文档解析中可中途停止，建议在错误数据上传后及时中止。
- 删除数据集/文档/切片均不可恢复。

### 2.6 技能管理
**功能描述**  
用于管理 Skills 列表，支持创建、查看详情、删除。

**操作步骤**
1. 进入`技能`页面，通过搜索框快速定位技能。
2. 点击`创建技能`打开创建弹窗并提交。
3. 点击技能卡片打开详情弹窗查看具体内容。
4. 需要移除时，点击删除按钮并确认。

**注意事项和常见问题**
- 创建与删除受权限控制，无权限时按钮不可用或执行失败。
- 删除技能前建议确认没有被关键流程引用。


### 2.7 工具管理（MCP）
**功能描述**  
用于管理 MCP 工具服务，支持创建、编辑、删除、导入、导出。

**操作步骤**
1. 进入`工具`页面，使用搜索框筛选目标工具。
2. 点击`创建工具`新增；点击卡片可进入编辑（有权限时）。

**注意事项和常见问题**
- 导出字段至少选择一个，否则无法导出。
- 删除后工具将不可被智能体继续调用。


### 2.8 模型服务商与系统模型
**功能描述**  
用于查看模型服务商状态，并配置系统默认模型。

**操作步骤**
1. 打开`模型服务商`页面，左侧选择目标服务商查看配置。
2. 点击右上`系统模型`打开配置弹窗。
3. 点击`保存`使系统默认模型生效。

**注意事项和常见问题**
- 语音相关模型入口已预留，当前为待定状态。


### 2.9 数据统计与监控
**功能描述**  
用于查看系统运行数据、Token 消耗趋势以及服务器健康状态。

**操作步骤**
1. 在`系统概况`查看用户、部门、角色、智能体、Token 等核心指标。
2. 点击部分统计卡片可直接跳转到对应管理页面。
3. 在`Token统计`中设置时间范围、部门、岗位后点击`查询`。
4. 查看趋势图、排行榜和明细表，分析消耗结构。
5. 在`服务器监控`点击`刷新`查看 CPU、内存、磁盘、Python 环境状态。

**注意事项和常见问题**
- 监控指标为实时快照，建议结合时间区间多次观察。
- Token 图表数据依赖统计接口，首次加载可能稍慢。


### 2.10 系统配置（用户/角色/菜单/组织等）
**功能描述**  
用于系统后台治理，包括用户、角色、部门、岗位、字典、参数、菜单等模块。

**操作步骤**
1. 在对应管理页使用筛选条件后点击`查询`，必要时`重置`条件。
2. 使用`新增`创建记录；在表格行执行`详情`、`编辑`、`删除`。
3. 需要批量处理时，先勾选数据，再执行批量启用/停用/删除。
4. 需要交换数据时，使用`导入`、`导出`功能（按模块支持情况）。
5. 在用户管理中可执行`重置密码`、上传头像、分配角色和岗位。
6. 在菜单管理中可维护路由、组件路径、权限标识、外链菜单和导出菜单配置。

**注意事项和常见问题**
- 部分内置对象（如默认管理员角色、超级管理员用户）不可删除或受限编辑。
- 菜单配置建议先在测试环境验证路由路径和组件路径，再发布到生产。
- 批量操作影响范围大，建议先筛选后执行。


## 3. 高级功能

### 3.1 角色权限分配
**功能描述**  
为角色分配可访问菜单与功能权限，实现按角色授权。

**操作步骤**
1. 进入`角色管理`，找到目标角色后点击`分配权限`。
2. 在权限抽屉中勾选菜单树节点（支持全选/取消）。
3. 点击`保存`完成授权。
4. 让该角色用户重新登录或刷新路由后生效。

**注意事项和常见问题**
- 系统默认角色通常不可操作。
- 权限分配后若页面仍不可见，先检查菜单状态与路由配置是否启用。

### 3.2 菜单与权限字治理
**功能描述**  
通过菜单管理维护前端路由、按钮权限与外链入口。

**操作步骤**
1. 进入`菜单管理`新增或编辑菜单。
2. 根据类型选择目录/菜单/按钮/外链，并填写对应字段。
3. 维护`权限标识`用于按钮级权限控制。
4. 使用`权限字`预览和`导出配置`进行统一治理与备份。

**注意事项和常见问题**
- 目录与菜单路由路径应以`/`开头。
- 菜单型项建议保证`路由名称`与页面组件命名一致。
- 错误的组件路径会导致页面无法正常加载。


### 3.3 知识库权限设置
**功能描述**  
在数据集级别设置可管理或只读权限，支持多人协作。

**操作步骤**
1. 在知识库左侧数据集列表中打开数据集操作菜单。
2. 点击`权限设置`，配置可管理用户/可访问用户。
3. 保存后，被授权用户可按权限访问数据集。

**注意事项和常见问题**
- 只读权限无法上传、删除、编辑文档与切片。
- 调整权限后建议让相关用户刷新页面确认权限变化。
