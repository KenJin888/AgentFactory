# AgentFactory - 用户操作手册

## 1. 简介

**AgentFactory 是一个面向企业与团队的 AI 智能体工作台，用于创建智能体、发起会话、管理知识库与后台权限。**  
系统将「智能体使用端」与「管理配置端」整合在同一前端，普通用户可快速进入会话与知识库，管理员可按角色配置菜单、权限和系统参数。  
页面支持动态菜单（由用户权限决定），因此不同账号看到的功能入口会不同。  

### 适用用户群体

- 业务用户：使用智能体广场、我的空间、会话、个人中心。
- 运营/知识库管理员：维护知识库、技能、工具、模型配置。
- 系统管理员：管理用户、角色、菜单、字典、参数、公告、日志与监控。

---

## 2. 系统登录与权限

### 登录方式

1. 打开系统登录页（`/#/login`）。
2. 在 **用户名** 输入框输入账号，在 **密码** 输入框输入密码。
3. 点击 **登 录 →**。
4. 登录成功后系统会跳转到首页（通常是 **我的空间**）。

### 忘记密码/注册流程

#### 注册

1. 在登录页点击 **马上注册**。
2. 填写 **用户名**、**密码**、**确认密码**。
3. 点击 **注 册 →**。
4. 注册成功后系统会自动登录并进入系统。

#### 忘记密码

1. 在登录页点击 **忘记密码?**。
2. 填写 **用户名**、**新密码**、**确认新密码**。
3. 点击 **重置密码 →**。
4. 页面提示重置成功后，返回登录页并使用新密码登录。

### 权限说明（必读）

- 菜单权限：控制左侧是否显示某个页面菜单。
- 操作权限：控制页面内按钮是否可见（如 **新增**、**编辑**、**删除**、**导出**）。
- 数据权限：部分模块按资源授权（例如知识库可按「只读/读写」区分）。
- 若角色菜单中某项 `status=1`（禁用），该项通常不会在前端作为可用菜单出现。

---

## 3. 界面总览

- **顶部区域**：页签栏（多会话标签）、当前页面标题与操作按钮。
- **左侧导航栏**：固定入口（如 **我的空间**、**智能体广场**）+ 动态管理菜单（按权限生成）。
- **主要内容区**：当前功能页（列表、卡片、表单、弹窗）。
- **右下用户区**：头像、用户名、帮助入口、退出登录按钮。
- **会话页专属区域**：消息列表、输入区（可含上传/联网/语音开关）、历史会话侧栏。

## 4. 核心功能操作指南

> 说明：当前仓库未包含 `backend/app/scripts/data/sys_menu.json` 实体文件，以下按前端真实页面与动态菜单机制整理；实际菜单层级以你的后端菜单配置与权限发布结果为准（并默认过滤禁用项）。

### 4.1 用户通用功能（优先）

#### 4.1.1 我的空间（`MySpace.vue`）

**模块用途与术语**

- 用于管理“我创建的智能体”、查看 Token 与会话统计、查看会话历史。
- 术语：  
  `智能体` = 可配置提示词/模型/工具/知识库的 AI 助手。  
  `会话` = 与某个智能体（或默认助手）的一次聊天线程。  

**操作步骤**

1. 在左侧点击 **我的空间**。
2. 点击右上 **创建智能体**（下拉可选 **新智能体** / **从文件导入** / **刷新页面**）。
3. 创建/编辑智能体时，在弹窗内依次配置：  
   **基础设定**（名称、描述、开场白、系统提示词、模型、封面）  
   **知识库**（勾选可用知识库）  
   **技能**（勾选技能）  
   **工具**（内置工具、MCP 工具）
4. 点击弹窗底部 **立即创建** 或 **保存修改**。
5. 在“我的智能体”卡片上可执行：打开会话、编辑、发布、删除。
6. 在“会话历史”表格点击任意行，可直接跳转到对应会话页。

**注意事项与常见问题**

- 点击智能体后显示“正在进入会话...”属于正常状态提示。
- 系统限制会话标签页数量，超限会提示“最多只能开启 N 个会话页签”。
- 从 JSON 导入智能体时，文件需包含必要字段（如 `name`、`prompt_template`）。

#### 4.1.2 智能体广场（`Agent.vue`）

**模块用途与术语**

- 浏览和使用已发布智能体，支持检索、分类过滤、收藏、复制与维护（有权限时）。

**操作步骤**

1. 打开 **智能体广场**。
2. 在顶部搜索框输入关键词，或点击分类标签筛选。
3. 点击 **收藏** 切换“仅看收藏”。
4. 点击某个智能体卡片直接进入新会话。
5. 点击卡片操作可执行：  
   **复制**（创建副本，部分场景可选“全新副本/可更新版本”）  
   **收藏/取消收藏**  
   **维护**（更新广场信息、授权、下线）
6. 在“管理广场智能体”弹窗中可配置：  
   **所有人**授权、**角色授权**、**用户授权**及权限级别（使用/可复制）。

**注意事项与常见问题**

- 收藏状态保存在用户偏好中，切换账号后收藏不互通。
- “下线智能体”后，广场卡片会移除，历史已创建副本不受影响。

#### 4.1.3 会话页（`ChatPage.vue`）

**模块用途与术语**

- 承载消息收发、模型切换、附件上传、语音输入、联网检索、历史会话切换。

**操作步骤**

1. 从 **我的空间** / **智能体广场** 进入会话，或点击侧栏 **打开会话** 新建。
2. 在顶部使用模型下拉（`ModelSelector`）切换当前会话模型。
3. 在输入区：
   - 输入文本后按 **Enter** 发送（`Shift+Enter` 换行）。
   - 点击 **回形针按钮** 上传附件（支持多格式）。
   - 点击 **地球按钮** 开启/关闭联网搜索（智能体开启该能力时可见）。
   - 点击 **麦克风按钮** 语音输入（智能体开启语音时可见）。
4. AI 响应中如需中断，点击 **停止按钮**。
5. 点击右上 **历史记录**，在侧栏选择历史会话或删除历史会话。

**注意事项与常见问题**

- 首条用户消息会自动用于更新会话标题（截取前 50 字符）。
- 某些能力按钮显示与否由智能体配置决定，不是异常。
- 语音输入依赖浏览器语音识别能力，不支持时会提示。

#### 4.1.4 个人中心（`Profile.vue`）

**模块用途与术语**

- 维护个人资料、头像与密码。

**操作步骤**

1. 点击左下角头像进入 **个人中心**。
2. 点击 **编辑资料**，修改 **姓名**、**手机号码**、**电子邮箱**，可点击头像区域更换头像。
3. 点击 **保存** 完成资料更新。
4. 点击 **修改密码**，填写 **当前密码**、**新密码**、**确认新密码** 后提交。
5. 需要退出时点击右上 **退出登录**。

**注意事项与常见问题**

- 头像仅支持 JPG/PNG，且大小不超过 1MB。
- 角色标签仅展示，不能在个人中心自行修改。

#### 4.1.5 帮助中心（`Help.vue`）

**模块用途与术语**

- 查询概念说明与产品指引，支持按分组浏览。

**操作步骤**

1. 点击侧栏或个人中心中的 **帮助中心**。
2. 在概念卡片页按分组展开/收起，点击任意卡片查看详情。
3. 在详情页可点击 **返回** 回到概览，或跳转到“相关概念”。
4. 点击右上 **关于我们** 查看版本与产品介绍。

**注意事项与常见问题**

- 帮助内容来自前端配置，更新后需发布前端版本才会生效。

### 4.2 AI 配置与资源管理

#### 4.2.1 知识库（`KnowledgeBase.vue`）

**模块用途与术语**

- 管理数据集、上传文档、文档解析、切片管理与检索测试。  
- 术语：  
  `数据集` = 一个知识库集合。  
  `文档` = 上传到数据集的原始文件。  
  `切片（Chunk）` = 文档解析后的最小检索单元。  

**操作步骤**

1. 进入 **知识库**，左侧选择一个数据集；没有数据集可点击 **+ 新建数据集**。
2. 数据集右侧菜单可执行：**权限设置**、**重命名**、**删除**（需管理权限）。
3. 在右上点击 **上传文件** 上传文档（支持多选）。
4. 在文档行可执行：  
   **查看切片**（放大镜）、**开始解析**、**停止解析**、**下载**、**重命名**、**删除**。
5. 点击 **检索测试**，输入问题并设置 `top_k`，可按文档过滤后点击 **检索**。
6. 在“切片管理”中可：刷新、新增切片、编辑切片、删除切片、展开/收起长文本。

**注意事项与常见问题**

- `只读` 权限账号可查看与检索，但不能上传、修改、删除。
- 文档状态 `RUNNING` 表示正在解析，可查看进度条并手动停止。

#### 4.2.2 技能（`SkillsSquare.vue`）

**模块用途与术语**

- 维护可被智能体调用的 Skills 列表。

**操作步骤**

1. 进入 **技能** 页面，使用搜索框按名称过滤。
2. 点击 **创建技能** 打开创建弹窗，填写并提交。
3. 点击技能卡片打开详情弹窗查看内容。
4. 在卡片右上点击 **删除** 删除技能（有权限时可见）。

**注意事项与常见问题**

- 删除技能后，依赖该技能的智能体需要重新检查配置。

#### 4.2.3 工具（MCP，`McpSquare.vue`）

**模块用途与术语**

- 管理 MCP 工具服务，支持导入导出。

**操作步骤**

1. 进入 **工具** 页面，使用搜索框过滤。
2. 点击 **创建工具** 新建，或点击卡片进入编辑。
3. 点击创建按钮下拉菜单可执行：  
   **导入**（上传 `.xlsx/.xls/.csv`）  
   **导出**（字段选择后导出）  
4. 卡片操作支持 **编辑**、**删除**（按权限显示）。

**注意事项与常见问题**

- 导入前建议先下载模板，字段不匹配会导致导入失败。
- 部分内置工具（如系统内置）不会在列表中展示为可管理项。

#### 4.2.4 模型服务商（`ModelProvider.vue`）

**模块用途与术语**

- 配置模型供应商与系统默认模型参数。

**操作步骤**

1. 打开 **模型服务商** 页面。
2. 在左侧选择供应商，右侧展示对应配置表单。
3. 根据供应商填写密钥、接口地址、模型列表等信息并保存。
4. 点击右上 **系统模型** 打开系统级模型配置弹窗并提交。

**注意事项与常见问题**

- 智能体创建前需至少有一个可用模型，否则会提示先配置模型。

#### 4.2.5 OpenHarness（`OpenHarness.vue`，调试工具）

**模块用途与术语**

- 用于调试会话 API：会话创建、查询、消息发送、流式消息与取消。

**操作步骤**

1. 打开 **OpenHarness** 页面，先点 **健康检查**。
2. 在“创建会话”填写可选参数后点击 **创建会话**，获取 `session_id`。
3. 在“会话操作”输入 `session_id`，执行 **查询会话** / **查询消息** / **取消会话**。
4. 在“普通发送”输入内容，点 **发送消息** 查看返回的 `assistant_text`、`usage`、`events`。
5. 在“流式发送”点击 **开始流式发送**，需要中断时点 **停止并取消**。

**注意事项与常见问题**

- 该页面偏向联调与排障，不建议普通业务用户日常使用。

### 4.3 系统配置与后台管理（按权限显示）

#### 4.3.1 用户管理（`UserManagement.vue`）

**模块用途与术语**

- 管理账号、角色绑定、状态与导入导出。

**操作步骤**

1. 打开 **用户管理**，在筛选区输入账号/用户名等条件后点 **查询**，点击 **重置** 清空条件。
2. 点击 **新增用户**，填写账号、用户名、密码、角色等后保存。
3. 行内操作可执行：**重置密码**、**详情**、**编辑**、**删除**。
4. 勾选多行后可执行：**批量删除**、**批量启用**、**批量停用**。
5. 点击工具栏 **导入** 上传 Excel，或 **导出** 导出当前/全部数据。

**注意事项与常见问题**

- 仅有“查询权限”的账号通常看不到新增/编辑/删除按钮。
- 超级管理员账号通常受保护，不允许被普通流程删除或勾选。

#### 4.3.2 角色管理（`RoleManagement.vue`）

**模块用途与术语**

- 管理角色本身、菜单权限与知识库权限。

**操作步骤**

1. 进入 **角色管理** 后可按角色名/状态查询。
2. 点击 **新增角色** 或行内 **编辑** 维护角色基础信息。
3. 点击行内 **分配权限**，在权限树勾选菜单后点 **保存**。
4. 点击行内 **知识库权限**，为角色选择数据集并设置 `只读/读写` 后保存。
5. 支持批量删除、批量启用/停用与导出。

**注意事项与常见问题**

- 菜单权限变更通常在重新登录后完整生效。
- 默认角色一般受保护，不可删除或修改关键授权。

#### 4.3.3 菜单管理（`MenuManagement.vue`）

**模块用途与术语**

- 维护动态菜单树、路由配置与权限字。

**操作步骤**

1. 打开 **菜单管理**，点击 **新增菜单**。
2. 在弹窗中先选 **菜单类型**（目录/菜单/按钮/外链），再填写对应字段：  
   **目录/菜单**：路由名、路由路径、可选组件路径  
   **按钮**：父级菜单、权限标识  
   **外链**：路由路径、外链目标地址
3. 通过行内按钮可 **新增子菜单**、**编辑**、**删除**。
4. 点击顶部下拉中的 **权限字** 可查看权限字列表。
5. 点击 **导出配置** 下载 `sys_menu.json`。

**注意事项与常见问题**

- 目录与菜单类型的路由路径需以 `/` 开头。
- 组件路径需与 `src/views` 下实际页面匹配。

#### 4.3.4 部门管理（`DeptManagement.vue`）与岗位管理（`PostManagement.vue`）

**模块用途与术语**

- 维护组织结构（部门）和岗位信息（岗位）。

**操作步骤**

1. 打开对应页面后先 **查询** / **重置**。
2. 点击 **新增部门/新增岗位** 新建记录。
3. 行内执行 **详情**、**编辑**、**删除**；部门支持 **新增子部门**。
4. 勾选多行执行 **批量删除**、**批量启用**、**批量停用**。
5. 岗位页面支持 **导出**；两页均支持 **刷新**、**列设置**。

**注意事项与常见问题**

- 删除前建议先确认是否已被用户或流程引用。

#### 4.3.5 字典管理（`DictManagement.vue`）

**模块用途与术语**

- 管理字典类型与字典数据（下拉项、枚举值）。

**操作步骤**

1. 在 **字典管理** 页面执行查询、重置。
2. 点击 **新增字典** 创建字典项。
3. 行内点击 **字典数据** 打开字典数据抽屉（`DictDataDrawer.vue`）维护具体枚举。
4. 行内支持 **详情**、**编辑**、**删除**；顶部支持批量与导出。

**注意事项与常见问题**

- 字典值被业务模块引用时，修改前需评估影响范围。

#### 4.3.6 参数管理（`ParamManagement.vue`）

**模块用途与术语**

- 维护系统参数（配置键值）。

**操作步骤**

1. 进入 **参数管理**，按参数名/键等条件查询。
2. 点击 **新增参数** 或行内 **编辑** 维护 `config_key/config_value` 等。
3. 行内可查看 **详情** 与 **删除**。
4. 支持 **批量删除**、**导出**、**列设置**、**刷新**。

**注意事项与常见问题**

- 配置键通常具有全局影响，建议变更前先记录旧值。

#### 4.3.7 公告管理（`NoticeManagement.vue`）

**模块用途与术语**

- 发布、维护系统公告内容。

**操作步骤**

1. 进入页面后通过筛选区 **查询** / **重置**。
2. 点击 **新增公告** 填写标题、内容、状态等信息后保存。
3. 行内执行 **详情**、**编辑**、**删除**。
4. 结合批量与导出能力完成日常运营维护。

**注意事项与常见问题**

- 公告内容建议先在测试环境预览，避免正式环境误发布。

#### 4.3.8 日志管理（`LogManagement.vue`）

**模块用途与术语**

- 查看系统日志记录（操作/行为审计）。

**操作步骤**

1. 打开 **日志管理**。
2. 使用筛选条件（用户、时间、类型等）执行查询。
3. 查看日志详情并结合条件复盘问题。

**注意事项与常见问题**

- 日志数据通常量大，建议先缩小时间范围再查询。

#### 4.3.9 服务器监控（`ServerMonitor.vue`）

**模块用途与术语**

- 实时查看服务器 CPU、内存、磁盘等监控信息。

**操作步骤**

1. 打开 **服务器监控**。
2. 查看系统资源卡片与磁盘明细。
3. 观察异常指标并配合日志页排查。

**注意事项与常见问题**

- 监控仅用于观察，不提供直接运维操作入口。

#### 4.3.10 Token 统计报表（`TokenAnalysis.vue`）

**模块用途与术语**

- 统计 Token 消耗与会话用量，支持运营分析。

**操作步骤**

1. 打开 **Token 统计报表**。
2. 按时间/用户/维度筛选查看统计结果。
3. 对高消耗会话回溯到对应业务页面优化提示词或模型配置。

**注意事项与常见问题**

- 统计数据可能存在刷新延迟，建议结合时间窗口复核。

#### 4.3.11 权限字总览（`PermissionPreview.vue`）

**模块用途与术语**

- 查看系统权限标识（权限字）全集，便于角色授权时参考。

**操作步骤**

1. 打开 **权限字总览** 页面，或在菜单管理中点击 **权限字**。
2. 按模块浏览权限字，确认角色需要授予的操作点。

**注意事项与常见问题**

- 建议按“最小权限原则”授权，避免一次性勾选过多写权限。
