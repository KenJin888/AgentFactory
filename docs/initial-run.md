# 🚀 项目初次运行指南

## 1. 登录

默认用户名密码：
- 用户名：admin
- 密码：123456

## 2. 配置模型

在 资源中心 - 模型管理 中，配置模型。

> 推荐 OpenAi、DeepSeek、硅基流动、Ollama，其他服务商未测试。


## 3. 创建第一个智能体
- 进入 我的空间 → 「创建智能体」
- 填写智能体名称，配置好智能体
- 保存后，点击智能体图标，进入对话窗口

## 4. 使用知识库
- 启用知识库功能要求后端的 .env
  - RAGFLOW_API_KEY：知识库 API 密钥
  - RAGFLOW_API_BASE：知识库 API 地址
