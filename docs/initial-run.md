# 🚀 项目初次运行指南

## 1. 登录

默认用户名密码：
- 用户名：admin
- 密码：123456

## 2. 配置模型

在 资源中心 - 模型 中，配置模型。

> 推荐 OpenAi、DeepSeek、硅基流动、Ollama，其他服务商未测试。

## 3. 运行第一个智能体

进入 智能体广场，点击「时间助手」智能体图标，进入对话窗口。
输入：“当前时间”
查看是否调用了工具，并返回当前时间。

## 4. 创建第一个智能体
**工厂创建**
- 进入 智能体广场，点击「Agent 工厂」图标
- 输入智能体需求
- 查看是否生成了智能体

**手工创建**
- 进入 我的空间 → 「创建智能体」
- 填写智能体名称，配置好智能体
- 保存后，点击智能体图标，进入对话窗口

## 5. 使用知识库
- 启用知识库功能要求后端的 .env
  - RAGFLOW_API_KEY：知识库 API 密钥
  - RAGFLOW_API_BASE：知识库 API 地址
